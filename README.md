# n8n-nodes-dingtalk-bby

包含1个钉钉触发器（信息、卡片）

## Prerequisites

开发环境:

* [git](https://git-scm.com/downloads)
* Node.js 和 npm. 最小版本 Node 20. 你可以使用Node版本管理工具NVM (Node Version Manager) for Linux, Mac, and WSL [点这里](https://github.com/nvm-sh/nvm). Windows用户看这个 [在Windows里安装NodeJS](https://docs.microsoft.com/en-us/windows/dev-environment/javascript/nodejs-on-windows).
* 安装 n8n:
  ```
  npm install n8n -g
  ```
* 推荐: 参考n8n安装指南 [设置开发环境](https://docs.n8n.io/integrations/creating-nodes/build/node-development-environment/).
